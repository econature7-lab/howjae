제2조 개인정보의 처리 방식

서버 미전송 원칙

생년월일 등 사주 분석에 사용되는 정보는 이용자의 기기(브라우저) 내에서만 처리됩니다. 어떠한 외부 서버로도 전송되거나 저장되지 않습니다.

localStorage 사용

관심 매물 기능은 브라우저 localStorage를 사용합니다. 이 데이터는 서버로 전송되지 않으며, 브라우저 설정에서 언제든지 삭제할 수 있습니다.

제3조 개인정보의 제3자 제공

사무소는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 다음의 경우는 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 의한 경우

제4조 개인정보의 파기

수집된 개인정보는 처리 목적이 달성되면 지체 없이 파기됩니다.

  • 사주 입력 정보: 브라우저 세션 종료 또는 페이지 이탈 즉시 삭제
  • 관심 매물(localStorage): 이용자가 브라우저 데이터 삭제 시 파기

제5조 이용자의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청
  • 개인정보 삭제 요청
  • 개인정보 처리 정지 요청
  • 본 서비스는 서버 저장 없이 운영되므로, 브라우저 데이터를 직접 삭제하시면 됩니다.

제6조 쿠키 및 외부 서비스

본 서비스는 다음 외부 서비스를 사용합니다.

  • Google Fonts: 폰트 로딩용 (Google 개인정보처리방침 적용)
  • Leaflet.js: 지도 표시용 오픈소스 라이브러리 (CDN 사용)
  • Firebase Firestore: 매물 데이터 저장용 (Google Firebase 개인정보처리방침 적용, 사주 정보 저장 없음)
  • 카카오 채널: 상담 문의용 (카카오 개인정보처리방침 적용)

제7조 공인중개사법 관련 사항

본 서비스의 사주 분석 기능은 참고용 정보 제공이며, 공인중개사법 제18조의2에 따른 중개대상물 표시·광고와 구분됩니다. 실제 부동산 거래는 반드시 공인중개사와 직접 상담하시기 바랍니다.

⚠️ 중요 안내

본 사이트의 사주팔자 운세 분석은 오락·참고 목적이며, 실제 투자 판단의 근거로 사용할 수 없습니다. 부동산 투자 및 거래에 관한 최종 결정은 본인의 책임으로 이루어져야 합니다.
